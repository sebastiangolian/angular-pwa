# angular-pwa
Test app with angular/pwa

## Test app 
[angular-pwa](https://sebastiangolian.github.io/angular-pwa)

## Getting started
```bash
npm install
```
```bash
npm run start
```

## Scripts
Build project
```bash
npm run build
```
Test project
```bash
npm run test
```
Test e2e project
```bash
npm run e2e
```
Lint project
```bash
npm run lint
```
Deploy files in gh-pages
```bash
npm run deploy
```
